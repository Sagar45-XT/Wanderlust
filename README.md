# Wanderlust

Wanderlust is a full-stack web application for discovering, listing, and reviewing unique places to stay around the world. Inspired by platforms like Airbnb, it allows users to create property listings, upload images, leave reviews, and explore locations on an interactive map.

## Features

- **User Authentication**: Sign up, log in, and log out securely.
- **Listings Management**: Create, edit, and delete property listings with image uploads.
- **Image Uploads**: Store listing images on Cloudinary.
- **Reviews & Ratings**: Leave and manage reviews for listings.
- **Interactive Maps**: View listing locations on Mapbox-powered maps.
- **Flash Messages**: User feedback for actions and errors.
- **Responsive UI**: Modern, mobile-friendly design using Bootstrap.

---

## Technologies Used

- **Backend**: Node.js, Express.js, MongoDB, Mongoose
- **Authentication**: Passport.js, passport-local-mongoose
- **Frontend**: EJS, EJS-Mate, Bootstrap, FontAwesome
- **File Uploads**: Multer, multer-storage-cloudinary, Cloudinary
- **Maps & Geocoding**: Mapbox, @mapbox/mapbox-sdk
- **Validation**: Joi
- **Session & Flash**: express-session, connect-mongo, connect-flash
- **Environment Variables**: dotenv
